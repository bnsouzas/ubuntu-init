# Ubuntu Init
Install apps and programming languages ​​on a fresh ubuntu 18.04 installation.

## Apps

### General

- Spotify

### Development

- Visual Studio Code
- DBeaver
- Redis Desktop Manager
- Postman
- Intellij Idea Community

## Programming Languages

- Java
  - Default Open JDK
  - Open JDK 8
- Nodejs
  - 10.x.x
- Python
  - 3.7.2
